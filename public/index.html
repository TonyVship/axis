Bubble is a panoramic rendering engine in JavaScript. It supports the rendering of equirectangular, cylindrical, and panoramic textures

Example


          var frame = new Bubble(document.querySelector('#my-video'), {
            autoplay: true,
            preload: true,
            height: 500,
            width: 500,
            src: '/path/to/video.mp4',
          });

          frame.render().on('ready', function () {
            // do stuff with player frame
          });
